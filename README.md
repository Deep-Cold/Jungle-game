# COMP3211 Group Project

## Play information

### In game system command

`saveGame [filename]` to save the game
`saveReplay [filename]` to save the replay
`close` to close the game without save

### In game play command

`drawback` to drawback your last move (including the opponent's) *Each player up to 3 times*
`PIECENAME DIRECTION` (pieceName can only be [rat, cat, dog, wolf, leopard, tiger, lion, elephant], direction can only be [U, D, L, R])

### Replay command

`next` to watch the next state
`prev` to watch the previous state
`quit` to end the replay

### main menu command

`startNewGame` to load a new game with default setting
`watchRelay` to check all replay that can view
`continueGame` to check all archive that can continue
`quit` to quit the program
